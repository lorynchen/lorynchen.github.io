Mindgrub
Work - Associate UX Designer

Mindgrub is an award-winning design agency based in Baltimore, MD. I interned here as an Associate during my last semester of college doing a number of user experience and user research projects.

During my time, I've worked with clients: NASA, Unisys Corporation, Johns Hopkins University, H.B. Fuller, and more.



Here is a brief summary of a few of the projects I worked on at Mindgrub.

NASA - Web Application

Designed an app that houses all code repositories scattered across the internet from NASA employees.

Johns Hopkins University - Web Application

Designed a website for a marketplace of books. Students go to rent books for class. Teachers go to assign books to students. Publishing companies go to sell books.

OPTAVIA - UX Audit

Audited their current web app of wellness training for consumers. Tested current flows and interactions as well as noted ways for improvement.

H.B. Fuller - Mobile Application

Designed a mobile app that acts as a resource for their distribution team to use for sales and troubleshooting information to consumers.

Wireframe Components Library - Internal System Library

Created a UI components library in Sketch to be used internally for wireframing. This included common UI elements for both iOS and Android as well as web components. Utilized InVision's Design System Manager product in their Craft Sketch Plugin.

Key Takeaways

I've gained so much experience working in an agency for 5 months. It was incredible to do impactful work with big name clients and to be able to work with such a talented team. I've developed my skills in user research, testing, time-management, client relations, Sketch, my craft, and my eye for good design.
